lf-ng-md-file-input

Usage

lf-ng-md-file-input directive will return a array via lf-files attribute binding, the object in array contain two object are lfFile and lfDataUrl, that you can grab file and preview dataurl easily.

Basic

<lf-ng-md-file-input lf-files="files01" ng-disabled="disabled01"></lf-ng-md-file-input>
Disable

Multiple

<lf-ng-md-file-input lf-files="files02" multiple></lf-ng-md-file-input>

Custom placeholder

<lf-ng-md-file-input lf-files="files03" lf-placeholder="my placeholder" ></lf-ng-md-file-input>

Preview

<lf-ng-md-file-input lf-files="files04" preview ></lf-ng-md-file-input>

Drag

<lf-ng-md-file-input lf-files="files05" drag preview ></lf-ng-md-file-input>

Combination

<lf-ng-md-file-input lf-files="files06" lf-placeholder="Pick Image" multiple drag preview ng-disabled="disabled06"></lf-ng-md-file-input>
Disable
